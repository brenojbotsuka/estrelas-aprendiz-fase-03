# Estrelas Aprendiz - Fase 03 ✨

Repositório centralizador de práticas e desafios da Fase 03 do programa de formação Estrelas Aprendiz. O objetivo deste espaço é consolidar o aprendizado backend de forma colaborativa e prática.

---

## 🛠️ Tecnologias Utilizadas

* **Linguagem:** Java 17
* **Framework:** Spring Boot
* **Gerenciador de Dependências:** Maven

---

## 🚀 Como Funciona a Dinâmica?

O fluxo de aprendizado é totalmente guiado e gerenciado pelas próprias alunas:

1. **Proposta:** Uma aluna sugere um novo exercício ou desafio criando uma **Issue** aqui no GitHub.
2. **Organização:** O acompanhamento do progresso das tarefas é feito utilizando o quadro **Kanban no GitHub Projects**.
3. **Execução:** Os códigos e resoluções dos exercícios propostos são integrados diretamente neste repositório.
